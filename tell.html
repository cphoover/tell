Tell.js

a bootstrapped reboot of apprise.js

Basically when looking for alternatives to native javascript alerts/prompts I came across Apprise.js by ThrivingKings. I really liked the way it looked and functioned, but when I wanted to extend it I realized all of the logic was all in one function. So I restructured the code, and switched the styles to use twitter bootstrap.

Bootswatch

Tell.js also works with any bootswatch theme. You can demo this by changing the stylesheet below:

Demo:

tell('Hello');
tell('Hello?', {'confirm':true});
tell('Hello now?', {'verify':true});
tell('How about now?', {'verify':true, 'textYes':'Yes already!', 'textNo':'No, not yet'});
tell('Hello', {'animate':true});
tell('What's your name?', {'input':true});
tell('Verify your name', {'input':'Jack Jones', 'textOk':'Correct'});
tell('Lorem ipsum...');
tell(document.getElementById('choices-template').innerHTML, {keyClose:false, showButtons:false});
tell(document.getElementById('login-template').innerHTML, {'verify':true, 'textYes':'Login', 'textNo':'Cancel', onOk: logIn});

Animate2 Compatibility

Tell.js is also made to work with animate2, my css3 transition animation library. animate2 uses css transitions instead of changing the style properties of an element on a set interval like the jquery animate function, and falls back to jquery animate if the browser doesn't support css3 transitons. Tell.js will use the default $.fn.animate function if animate2 is not available.
